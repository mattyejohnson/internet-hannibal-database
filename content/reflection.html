Reflection

A reflection on using Bulma.

I chose Bulma to use as the CSS framework for this assignment. I chose it because I have never used it before, and because I like how the examples look in the documentation. Implementing it was easy, and I found that it offers a diverse amount of options for webpage elements such as navbars, columns, tables, cards, and otherwise. I particularly like the built-in responsiveness. If I find myself in need of a CSS framework in the future, I will likely use this one again.
